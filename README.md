## Freemind Mindmap Formatter

Export [Freemind][0] mindmaps.

[0]: http://freemind.sourceforge.net/wiki/index.php/Main_Page

## Scope
### Out of scope

  * HTML nodes

### Features

  * Specify a formatter from the command-line
  * Formatters are specified as Python programs
